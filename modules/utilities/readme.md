# Usonian CSS Utilities

Contains classes used for setting low-level stuctural / positional traits applicable to any element.

## Accessibility

**Available Classes**

```
.u-visuallyHidden # Visually hide element but make it available for screen readers
```

## Clearfix

**Available Classes**

```
.u-clearfix # Clear both sides of an element, useful when using floats
```

## Color

**Available Classes**

```
.u-white
.u-accent
.u-grayLightest
.u-grayLighter
.u-grayLight
.u-gray
.u-grayDark
```

## Display

**Available Classes**

```
.u-block # Sets display value to "block"
.u-flex # Sets display value to "flex"
.u-inline # Sets display value to "inline"
.u-inlineBlock # Sets display value to "inline-block"
.u-inlineFlex # Sets display value to "inline-flex"
.u-none # Sets display value to "none"
```
